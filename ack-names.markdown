# What do we call ack 2.0?

Must be

* pronouncable
* easy to type
* three characters
* relatively low Googling conflict
    * not already a word
* No conflict with existing BSD ports, Ubuntu packages or Fedora RPMs

http://www.acronymfinder.com/ is handy
